# 📸 Image Placeholder Instructions

Folder `assets/images/` adalah tempat untuk menyimpan gambar portfolio Anda.

## Gambar yang Dibutuhkan

Siapkan **4 gambar** dengan spesifikasi:

| Nama File | Untuk | Ukuran Min | Format |
|-----------|-------|-----------|--------|
| `project-1.jpg` | Mobile App Design (UI/UX) | 600x600px | JPG/PNG |
| `project-2.jpg` | Website Redesign (UI/UX) | 600x600px | JPG/PNG |
| `project-3.jpg` | Sales Dashboard | 600x600px | JPG/PNG |
| `project-4.jpg` | Customer Analytics | 600x600px | JPG/PNG |

## Langkah-Langkah Menambah Gambar

### 1. Siapkan Gambar
- Screenshot dari project Anda
- Atau design mockup
- Minimal 600x600px (lebih besar semakin baik, tapi optimize size)

### 2. Optimize Gambar
Gunakan salah satu tools online:
- **TinyPNG.com** - Best! Compress dengan smart lossy
- **Compressor.io** - Manual compression
- **Squoosh.app** - Google's tool, very good
- **ImageOptim.com** - Simple compression

Target: < 200KB per gambar untuk fast loading

### 3. Rename Gambar
Rename file dengan nama yang tepat:
```
project-1.jpg  ← Mobile App Design
project-2.jpg  ← Website Redesign
project-3.jpg  ← Sales Dashboard
project-4.jpg  ← Customer Analytics
```

### 4. Copy ke Folder Ini
Copy file ke: `Portofolio/assets/images/`

### 5. Verifikasi
- Buka `index.html` di browser
- Scroll ke portfolio section
- Gambar harus muncul dengan smooth animation

## Tips

✅ **Gunakan Screenshot Tools**
- Windows: Win+Shift+S (Built-in Snip & Sketch)
- Mac: Cmd+Shift+4
- Chrome: DevTools → ... → Capture screenshot

✅ **Dimensi Recommendations**
- Square: 600x600px, 800x800px, 1000x1000px (best)
- Responsive: Website akan auto-scale

✅ **Quality Tips**
- Screenshot at 125% zoom untuk sharp hasil
- Gunakan design mockups (Figma, Sketch, XD)
- Show actual work, bukan random images!

✅ **Optimization Checklist**
- [ ] Compress gambar (target < 200KB)
- [ ] Crop ke square 600x600px
- [ ] Rename ke project-1, project-2, dll
- [ ] Verify di browser (tidak broken images)

## Jika Belum Ada Gambar Real

### Option 1: Placeholder Templates
Buat di Figma atau Canva dengan:
- Project title
- Description
- Tools/technologies used
- Your photo/logo

### Option 2: Design Mockups  
- Gunakan mockup templates dari designcapital.com
- Edit dengan Figma/Canva
- Export sebagai gambar

### Option 3: Temporary Placeholder
Untuk testing, bisa gunakan:
- unsplash.com (free high-quality)
- pexels.com (portfolio images)
- pixabay.com (generic images)

Nanti bisa diganti dengan project real Anda.

## Troubleshooting

❌ **Gambar tidak muncul?**
- Cek nama file (case-sensitive di Linux/Mac)
- Cek path: `assets/images/project-1.jpg`
- Check browser console (F12) untuk error
- Clear cache (Ctrl+Shift+R)

❌ **Gambar blur/pixelated?**
- Ukuran asli terlalu kecil
- Gunakan gambar minimal 600x600px
- Atau buat mockup lebih besar

❌ **Loading lambat?**
- File terlalu besar
- Compress menggunakan TinyPNG
- Target max 200KB per file
