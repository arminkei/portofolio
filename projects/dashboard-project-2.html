Farmasi Overview Dashboard

Ringkasan performa operasional dan penjualan jaringan farmasi secara menyeluruh

Gambaran Umum

Dashboard ini menyajikan ringkasan performa operasional dan penjualan jaringan farmasi secara menyeluruh. Fokus utamanya adalah memberikan visibilitas terhadap data transaksi, persebaran geografis cabang, dan kontribusi kategori produk obat serta vitamin guna mendukung pengambilan keputusan manajemen.

Tujuan Proyek

  • Analisis Tren Penjualan: Memantau fluktuasi pendapatan bulanan dan volume transaksi untuk mengidentifikasi pola musiman
  • Pemetaan Wilayah: Menganalisis distribusi penjualan berdasarkan provinsi untuk mengidentifikasi wilayah dengan kontribusi tertinggi
  • Segmentasi Produk: Mengevaluasi performa kategori produk guna mengoptimalkan manajemen stok di berbagai cabang
  • Retensi Pelanggan: Mengukur konsistensi jumlah pembeli aktif setiap bulan untuk memahami loyalitas pasar

Metodologi

1

Data Integration

Integrasi data dari semua cabang farmasi

2

Data Warehousing

Struktur data di BigQuery untuk analisis

3

Geographic Analysis

Mapping penjualan per region & provinsi

4

Dashboard Publishing

Publikasi di Looker Studio untuk stakeholders
