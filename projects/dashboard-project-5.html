Robotics Sales Report

Dashboard analisis penjualan robotika dan edukasi teknologi 2020-2021

Gambaran Umum

Laporan ini merupakan dashboard analisis penjualan untuk sektor robotika dan edukasi teknologi yang mencakup rentang waktu tahun 2020 hingga 2021. Laporan ini memberikan gambaran makro mengenai pendapatan, identifikasi pelanggan utama, serta perbandingan performa antara produk fisik dan digital.

Tujuan Proyek

  • Monitoring Metrik Utama: Melacak KPI inti seperti total pesanan, Average Order Value (AOV), dan total pendapatan tahunan
  • Analisis Geografis: Memetakan distribusi pelanggan di berbagai wilayah untuk mengidentifikasi pasar regional terkuat
  • Evaluasi Produk: Menganalisis kontribusi pendapatan berdasarkan kategori produk seperti robot, drone, dan materi edukasi
  • Identifikasi Pelanggan Loyal: Menampilkan daftar pelanggan dengan nilai transaksi tertinggi untuk mendukung strategi retensi

Metodologi

1

Sales Data Extraction

Ekstraksi data penjualan 2020-2021

2

Customer Segmentation

Segmentasi berdasarkan purchasing behavior

3

Product Performance

Analisis kontribusi setiap kategori produk

4

Report Generation

Pembuatan interactive report di Looker Studio
