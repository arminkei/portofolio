Pizza Sales Performance Dashboard

Visualisasi performa bisnis penjualan pizza sepanjang satu tahun

Gambaran Umum

Proyek ini merupakan visualisasi data komprehensif yang menyajikan performa bisnis penjualan pizza sepanjang satu tahun. Dashboard ini dirancang untuk memudahkan pemangku kepentingan dalam memahami metrik operasional seperti total pesanan, pendapatan, dan tren menu terlaris dalam satu tampilan yang bersih dan modern.

Tujuan Proyek

  • Monitoring Real-Time: Menyediakan visualisasi pendapatan bulanan untuk memantau fluktuasi penjualan sepanjang tahun
  • Analisis Produk: Mengidentifikasi menu paling menguntungkan dan rincian volume per kategori produk
  • Optimalisasi Inventaris: Menganalisis korelasi antara ukuran produk dan efisiensi pendapatan untuk strategi stok bahan baku
  • Peningkatan Kepuasan Pelanggan: Memahami perilaku pembelian melalui Average Order Value (AOV) untuk menyesuaikan strategi harga

Metodologi

1

Data Extraction

Ekstraksi data penjualan pizza dari sistem POS

2

Data Modeling

Transformasi dan struktur data di BigQuery

3

Dashboard Design

Visualisasi intuitif di Looker Studio

4

Deployment

Testing dan deployment ke production
